# Arquivos de Exemplo - Sistema de Controle de Material

Esta pasta contém arquivos de exemplo para testar o sistema.

## 📄 Tipos de Arquivos

### Notas de Empenho (Exemplo)

- Formato: PDF
- Conteúdo: Dados estruturados de empenho
- Uso: Teste da funcionalidade de upload e extração

### Notas Fiscais (Exemplo)

- Formato: PDF
- Conteúdo: NF-e ou nota fiscal comum
- Uso: Teste da comparação com empenhos

## 🧪 Como Testar

1. Use os arquivos desta pasta para testar o upload
2. Verifique se os dados são extraídos corretamente
3. Teste a comparação entre NF e NE
4. Valide os relatórios gerados

## ⚠️ Observações

- Os arquivos de exemplo são fictícios
- Dados pessoais e valores são simulados
- Use apenas para fins de teste e demonstração
- Substitua por arquivos reais em produção

## 📝 Estrutura Esperada nos PDFs

### Nota de Empenho deve conter:

- Número do empenho
- Data
- Fornecedor
- CNPJ
- Lista de itens com códigos, descrições, quantidades e valores

### Nota Fiscal deve conter:

- Número da NF
- Data de emissão
- CNPJ do emissor
- Chave de acesso (se NF-e)
- Lista de itens
