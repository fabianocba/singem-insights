Como funciona

Operacao em Banco/API

Empenhos, Notas Fiscais e anexos são persistidos no banco de dados.

Não há configuração de pastas locais neste modo.

3. Estrutura Automática:

O sistema cria/detecta a seguinte estrutura:

[Pasta Escolhida]/
  └─ IF Guanambi/ (abreviação da sua unidade)
      └─ 2024/ (ano atual)
          ├─ Notas de Empenho/
          └─ Notas Fiscais/

4. Organização Inteligente:

  • Ano automatico: documentos sao salvos na pasta do ano de emissao
  • Pastas sob demanda: anos anteriores sao criados quando necessário
  • Nomeacao inteligente: arquivos recebem nomes unicos automaticamente
  • Compativel com manual: reconhece pastas criadas manualmente

5. Exemplo Prático:

NE emitida em 15/03/2023
➜ Salva em: IF Guanambi/2023/Notas de Empenho/NE_2023_001.pdf
NF emitida em 20/08/2024
➜ Salva em: IF Guanambi/2024/Notas Fiscais/NF_2024_123.pdf
Avisos importantes
Permissoes do navegador:
• Use Chrome, Edge ou Opera atualizado
• O sistema solicitará permissão para acessar a pasta escolhida
• Esta permissão é necessária para salvar os documentos automaticamente
Armazenamento local:
• Os arquivos ficam salvos APENAS no seu computador
• Faça backup regular da pasta configurada
• A configuração fica salva no navegador (IndexedDB)
Gerenciamento e protecao de arquivos
Importante - protecao de dados:
• NÃO exclua arquivos pelo Windows Explorer!
• Exclusões externas desvinculam o arquivo do registro no sistema
• Use sempre as ferramentas abaixo para manter a integridade dos dados

Sincronizacao de arquivos

Detecta arquivos que foram deletados externamente e atualiza os registros do banco de dados.

Verificar integridade

Verifica se todas as pastas configuradas existem e se foram deletadas externamente.

Reparar estrutura

Recria pastas que foram deletadas externamente, restaurando a estrutura completa.

Limpar registros deletados

Exclui permanentemente do banco de dados os registros de empenhos e notas fiscais cujos arquivos foram deletados externamente.

Limpar registros orfaos

Exclui permanentemente registros de empenhos/NFs que NÃO possuem arquivo PDF vinculado. Use para limpar dados inconsistentes ou cadastrados incorretamente.

Como funciona:

  • Sincronização Automática: Executa ao iniciar o sistema
  • Detecção de Exclusões: Identifica arquivos deletados no Explorer
  • Atualização Inteligente: Marca registros com "arquivo deletado"
  • Reparo Automático: Recria pastas faltantes mantendo a estrutura
  • Log Completo: Todas as operações são registradas no console
